# Full-Stack CI/CD App

Task Manager с полным CI/CD pipeline.

## Компоненты
- **Frontend**: HTML/CSS/JS (nginx)
- **Backend**: Flask REST API
- **Database**: PostgreSQL

## Эндпоинты API
- GET /api/data
- POST /api/data
- DELETE /api/data/:id

## CI/CD
- CI: GitHub Actions (pytest)
- CD: Railway (автодеплой)
